# AstralNET
Проект состоит из 3 модулей
- **telegram-bot** (Телеграм Бот)
- **miniapp** (Мини приложение для телеграмма)
- **fast-api** (Бэкенд)

Требуется редактирование файла <code>config.py.example</code> в <code>config.py</code> и соотвественно настроить его для корректной работы.

Telegram Bot:
```bash
python -m telegram-bot.init
```
